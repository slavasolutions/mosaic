Mosaic — a folder shape for web content

Files are records. Folders are collections. One reserved file (mosaic.json). Refs link records via cascade lookup. Expand folders below to see all the cases discussed in 0.9.

STAC — SpatioTemporal Asset Catalog

Apache 2.0 spec for satellite imagery and earth observation. Same shape as Mosaic — directory tree of JSON + binary assets — but vocabulary tuned for geo data. Typed rel links between nodes (parent / child / self / derived_from). The closest structural analog to Mosaic; worth reading end-to-end.

OCI Image Layout — container images on disk

Apache 2.0 spec from the Open Container Initiative. How Docker / Podman / containerd persist images. One JSON manifest at root, content-addressed binary blobs in blobs/sha256/. Each blob is fingerprinted by its SHA-256 hash — rename-safe by design.
