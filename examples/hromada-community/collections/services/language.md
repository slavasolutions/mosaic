# Language classes

We run Ukrainian classes for Canadian-born children of Ukrainian heritage, and English classes for recent newcomers.

## Schedule

- **Saturday Ukrainian** — children ages 5-12, 9am-12pm
- **Adult Ukrainian conversation** — Wednesdays, 7-9pm
- **Beginner English** — Tuesdays and Thursdays, 6-8pm
- **Intermediate English** — Mondays and Wednesdays, 6-8pm

## Cost

Adult classes are free. Children's classes are pay-what-you-can.
