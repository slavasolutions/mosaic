# Examples

Reference Mosaic documents. Document-only — no rendered output, no engine-specific code.

| Example | What it shows |
| --- | --- |
| [`minimal-site/`](minimal-site/) | Smallest meaningful Mosaic document |
| [`blog-site/`](blog-site/) | Collection with `ref:` resolution and markdown body assets |

Each example is a valid Mosaic document that conforms to the current `spec.md`. A reader can validate any of them and produce output.

## Contributing examples

Examples should:

- Be small enough to read end-to-end in a few minutes
- Demonstrate one or two specific format features clearly
- Not depend on any particular reader or rendering engine
- Include a short README explaining what the example demonstrates

Examples should not:

- Include rendered output (HTML, screenshots, demos)
- Include engine-specific code (Astro components, CLI invocations)
- Depend on Clear or any other implementation
